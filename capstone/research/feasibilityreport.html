Feasibility Report

Features of capstone project:

  • Enable users to create posts

    Technologies required: JavaScript and back4app

    I will use JavaScript with back4app to allow users to submit text messages with a form.

  • Enable users to like or dislike posts

    Technologies required: JavaScript and back4app

    I will use JavaScript to create event listeners for clicks on like and dislike buttons which will then be added to the sum and be sent to back4app. The data about likes and dislikes will determine the area of the post.

  • Visualize the posts into a bubble chart

    Technologies required: JavaScript, back4app, and D3 library

    I’m not yet skilled at making a bubble chart with D3, but I have found an example on the library website that could help me to create one. In case I don’t manage to do so, the visualization can be easily simplified into a grid system which would only require knowledge about CSS and JavaScript.
