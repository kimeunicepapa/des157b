Arrow functions

The function calculates the sum of two numbers. I've created this function for this exercise.

Named Function

            function totalNumber(x, y){
                const sum = x+y;
                return `the sum is ${sum}`;
            }

Function Expression

            const totalNumber = function(x, y) {
                const sum = x + y;
                return `the sum is ${sum}`;
            };

Arrow Function

            const totalNumber = (x, y) => {
                const sum = x + y;
                return `the sum is ${sum}`;
            };
